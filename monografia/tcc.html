Projeto

Sistema de Gerenciamento de Competências - SIGECO (Resumo da Monografia)

Baseado no artigo "Data, Information, Knowledge and Competence" do Professor Valdemar W.Setzer foi idealizado um software para organizar de forma dados sobre conhecimentos e competências de profissionais de qualquer área.

As competências são representadas em células de matrizes bidimensionais, denominadas Matrizes de Competência, onde as linhas contêm áreas de conhecimento, e as colunas as habilidades que se aplicam a essas áreas.

Dadas as matrizes de competência do sistema é possível fazer com que pessoas cadastrem para cada matriz as suas competências pessoais. Formamos assim um Banco de Dados de competências onde estes dados podem ser cruzados para encontrar pessoas que tenham determinadas competências.

O software tem várias aplicações diretas, em particular:
- Auxiliar o gerenciamente de Recursos Humanos de grandes empresas.
- Agregar valor a sites de currículos online para que empresas possam procurar por novos candidatos com maior precisão.
- Possibilitar o contato entre profissionais, permitindo a localização de pessoas que detêm certos conhecimentos ou competências.

Proposta (Objetivo)

A proposta de TCC é desenvolver o software SIGECO em Java para a plataforma Web. Queremos adotar diversas tecnologias Java existentes hoje em dia para formar um ambiente de desenvolvimento ágil.

Atividades já realizadas

- Discutimos a modelagem conceitual de dados inicial acrescentando uma nova entidade possibilitando Sub-Grupos de Conhecimentos em n-níveis.
- Criamos a representação no computador do modelo conceitual básico por meio de classes.
- Criamos o banco de dados inicial da aplicação usando ferramentas de mapeamento objeto-relacional.
- Criamos algumas telas iniciais para o sistema (protótipos)
